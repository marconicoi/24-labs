## live-preview
CDN
```
https://cdn.jsdelivr.net/gh/marconicoi/24-labs@main/jquery.live-preview.js
```
Usage
```
<input id="fname' />
<div data-24-bind="#fname"></div>
```
or
```
<input id="a" />
<input id="b" />
<div data-24-calc="#a+#b"></div>
```
or
```
<input type="checkbox" id="make-visible' />
<div data-24-show="#make-visible"></div>
```
